# 🩺 Docteur ProConnect

Docteur ProConnect is a tool to inspect ProConnect authentication flows and the data shared with partner services.

## 🐇 Run it with Bun

Install the dependencies:

```
bun install
```

Run the server:

```
bun start
```

## 🧪 Tests

```
bun test
```

## ✨ Format

```
bun run format
```
